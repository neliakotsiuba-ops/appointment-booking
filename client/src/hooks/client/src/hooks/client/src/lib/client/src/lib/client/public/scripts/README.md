# 📅 Appointment Booking System

A modern, mobile-friendly PWA for scheduling appointments — built with React, TypeScript, and Tailwind CSS.

## ✨ Features

- 📆 Date selection for the next 14 days
- ⏰ Real-time available time slot loading
- 📋 3-step booking flow with live summary sidebar
- 🌙 Automatic dark mode support
- 📱 Fully responsive — works on mobile and desktop
- ⚡ Installable as a PWA (Progressive Web App)

## 🛠 Tech Stack

- **React 18** — UI library
- **TypeScript** — type-safe code
- **Tailwind CSS** — utility-first styling
- **Vite** — fast build tool
- **shadcn/ui** — accessible UI components
- **Node.js / Express** — backend API
- **SQLite** — lightweight database

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/your-username/appointment-booking.git
cd appointment-booking
npm install
npm run dev
```

The app will be available at `http://localhost:3000`
